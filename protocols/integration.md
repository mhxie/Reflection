# Integration Protocol

Connects insights to action. The gap between "I realized X" and "I changed Y" is where most reflection value is lost. This protocol bridges it.

## The Insight-to-Action Pipeline

```
Insight → Capture → Connect → Commit → Check
```

### 1. Capture
Every insight surfaced during a session must be:
- Written down (in reflection file or Reflect note)
- Attributed (where did it come from? [[Note]] or session observation?)
- Timestamped

### 2. Connect
Link the insight to:
- A specific goal (which goal does this serve?)
- A specific behavior (what would change if you acted on this?)
- A specific timeline (when is this relevant?)

### 3. Commit
Convert insight to commitment:

| Insight Level | Commitment Type | Example |
|---|---|---|
| Observation | Note it | "I notice I avoid health goals" — no action needed yet |
| Connection | Explore it | "Career + learning goals overlap" — think about it this week |
| Pattern | Address it | "I consistently neglect X" — schedule a specific action |
| Tension | Resolve it | "Goal A conflicts with Goal B" — make a decision |
| Implication | Act on it | "If this pattern continues, Y" — change behavior now |

### 4. Check
In the next session, verify:
- Was the commitment acted on?
- If yes, what happened?
- If no, why not? (This is data, not judgment)

## The Accountability Question

At the start of every session after the first, ask:
"Last time, you intended to [action from previous session]. How did that go?"

Three possible responses and how to handle them:

| Response | Meaning | Next Step |
|----------|---------|-----------|
| "I did it" | Follow-through works | Celebrate briefly, then ask what came of it |
| "I partially did it" | Good signal | Explore what enabled the partial + what blocked the rest |
| "I didn't" | Not judgment — data | Explore without blame: "What got in the way?" |

## Integration Failure Patterns

| Pattern | Signal | Intervention |
|---------|--------|-------------|
| **Insight tourism** | Great realizations, no behavior change | Ask: "What would you do differently starting tomorrow?" |
| **Commitment inflation** | Too many next actions, none completed | Limit to ONE next action per session |
| **Accountability avoidance** | Changes topic when asked about follow-through | Note gently, don't push |
| **Action addiction** | Always needs an action item, never sits with ambiguity | Allow "just notice" as valid commitment |

## Rules

1. **One action per session.** Don't overwhelm with multiple commitments.
2. **Specific beats vague.** "Think about my career" is not actionable. "Write a note about what success looks like in my new role by May 1" is.
3. **Small beats ambitious.** A 5-minute action done is worth more than an ambitious action deferred.
4. **Sitting with it counts.** Not every insight needs an action. "Notice when X happens" is a valid commitment.
5. **No guilt.** Missed actions are data points, not failures.
